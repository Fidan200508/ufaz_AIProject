Starter-Pack Policy
The starter pack is intentionally minimal. It includes:

fixed digits features, labels, and split indices
deterministic split and synthetic-data scripts
a minimal repository skeleton
a short README and checklist
an optional report template
It does not include model code, training-loop scaffolding, optimizer implementations, or hidden solutions.

Suggested First Steps
Review the starter-pack README and checklist.
Clone the repository and create your working branches.
Inspect the provided data files before writing model code.

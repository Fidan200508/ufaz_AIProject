# Source Code Directory (`/src`)

This directory is designated for the core implementation of the AI project.

## 📋 Overview
The `src` directory contains all the foundational scripts and modules required for the capstone project. In alignment with the project requirements, this directory starts as an empty structure in the starter pack to ensure that teams implement the learning algorithms from scratch.

## 🛠 Required Implementations
At a minimum, the final repository should include implementation files for the following components:

* **Data Loading:** Utilities to parse and preprocess the datasets.
* **Softmax Regression:** Implementation of multi-class logistic regression.
* **Neural Network (One-Hidden-Layer):** Core architecture using `tanh` or `ReLU` activations.
* **Training Loops:** Functions for forward passes, backpropagation, and weight updates.
* **Evaluation Utilities:** Tools to calculate accuracy, loss, and other metrics.
* **Plotting Helpers:** Visualizing decision boundaries, loss curves, and confusion matrices.

## ⚠️ Important Note
**Do not replace your own implementation with framework code.** The goal of this capstone is to demonstrate a deep understanding of the mathematical foundations of AI. Therefore, the use of high-level deep learning frameworks (like TensorFlow, Keras, or PyTorch) for core logic is strictly prohibited unless specified otherwise. All algorithms must be implemented using fundamental libraries like `NumPy`.
